Inside Out
A personal website which also exposes some of my personal information. Intended to be an ongoing project, it currently displays my SMS, Call, Calendar and debit card transaction data. Why? A little bit of an experiment to see how it feels...
Feb 2017
Bitip
An experiment to see if I could build an app that allows for direct tipping to artists. It used a blockchain api alongside web sockets to confirm the transactions.

Club Mate
A group project where we all agreed that it would be great to have an app which would allow for us to find other people who share similar interests and go to events with them. Log in and have a look around - not too bad for a 4 day hack!

Monzo Heatmap
Using the Monzo api I built a system which shows the user their transactions plotted on a map. If you have a monzo card you can try yourself - just go to the dev tools area at monzo to get some credentials (the link is on the homepage of the app)
